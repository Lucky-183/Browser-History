# 浏览器历史记录获取器

<br>

## 功能

- 获取主流浏览器的历史记录，保存为xls文件
- 对于主流搜索网站，提取搜索关键词

<br>

## 使用方法

### Way1：直接使用

下载EXE文件后，打开程序，选择浏览器（默认为默认浏览器），点击按钮即可在当前文件夹下生成.xls浏览器历史记录文件。

### Way2：源代码

在目录下运行 ```main.py```

<br>

## 注意

- 由于浏览器的历史记录存储文件有空间限制（大约10000条），所以可能无法获取到较早的浏览器历史记录。
- 本程序能够与上次生成的.xls文件比对，保留完整的浏览器历史记录，所以有需要可以定期运行。
